# Prompt Logic Documentation

## Project

AI SEO Blog & Content Cluster Generator

## Business Selected

Bright Smile Dental Clinic

---

# Prompt 1: SEO Blog Generation

## Initial Prompt

Write a blog about dental care.

### Issue

The content was too generic and lacked SEO structure.

## Improved Prompt

Write a 1200-word SEO-optimized blog titled:
"Top 10 Dental Care Tips for Healthy Teeth and Gums"

Include:

* H1, H2, H3 headings
* SEO-friendly keywords
* Conclusion
* Call-to-action

### Result

Generated a well-structured SEO blog suitable for business websites and search engines.

# Prompt 2: Content Cluster Generation

## Initial Prompt

Generate dental blog ideas.

### Issue

The ideas were not organized into a content strategy.

## Improved Prompt

Create a content cluster for a dental clinic website with one pillar topic and multiple supporting articles.

### Result

Generated a content cluster that supports SEO growth and internal linking.

# Prompt 3: Local SEO Strategy

Generate local keywords for a dental clinic.

Generate local SEO keywords and blog ideas targeting Vijayawada.

### Result
